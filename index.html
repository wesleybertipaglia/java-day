Java Day

26 de Agosto de 2023

Estamos ansiosos para o próximo Java Day, organizado pela equipe do Java Noroeste! O evento promete trazer uma programação repleta de palestras e workshops, abordando tópicos atuais e relevantes no universo do Java. Será uma oportunidade única para os participantes se atualizarem sobre as mais recentes tendências e práticas da linguagem. Além disso, o evento oferecerá uma plataforma de networking e troca de experiências entre os entusiastas e profissionais da área. Prepare-se para um dia repleto de aprendizado e conexões no próximo Java Day!

Av. dos Estudantes, 3278 - Jardim Novo Aeroporto, São José do Rio Preto - SP (Ver no Maps) 26 de Agosto de 2023 09h - 17h

Palestrantes

Gabriel Silva Andrade O poder da comunidade

O mercado exige que tenhamos nossas hard e soft skills cada vez mais afiadas, e por que não aprender com grandes referências do desenvolvimento de software mundial? Mas como estar em contato com essas referências? É aqui que entra o poder da comunidade! Com ela você terá oportunidades de desenvolver suas hard e soft skills de maneira colaborativa, desenvolvendo um valioso networking, potencializando assim sua carreira! Nesta palestra mostrarei como alcançar essas possibilidades de crescimento de carreira

Dorival Querino da Silva Microsserviços com Java na Prática com Spring Boot e Spring Cloud

Nesta apresentação serão abordados os conceitos de Microsserviços, bem como suas vantagens e desvantagem. A seguir será mostrado uma implementação do zero utilizando Spring, mostrando na prática a aplicação dos conceitos apresentados.

Yugo Sakamoto 4 conceitos de arquitetura que todo desenvolvedor precisa conhecer

A arquitetura de software existe para ajudar no design e construção de uma solução, e conhecer as práticas dessa disciplina podem te tornar um melhor desenvolvedor. Nessa apresentação passarei pelos princípios essenciais para te guiar numa carreira de sucesso!
